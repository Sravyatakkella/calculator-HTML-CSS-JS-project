🧮 Calculator App

A simple and responsive Calculator Application built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations with a clean and user-friendly interface.

📌 Features
➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division
🧹 Clear (AC) functionality
⌫ Delete last digit
📱 Responsive design for desktop and mobile
⚡ Fast and lightweight
🛠️ Technologies Used
HTML5 – Structure
CSS3 – Styling and layout
JavaScript (ES6) – Calculator logic and event handling
📂 Project Structure
calculator-app/
│
├── index.html      # Main HTML file
├── style.css       # Stylesheet
├── script.js       # Calculator functionality
└── README.md       # Project documentation
🚀 Getting Started
1. Clone the repository
2. Navigate to the project folder
cd calculator-app
3. Open the project

Simply open index.html in your web browser.

🎯 How to Use
Enter numbers using the calculator buttons.
Select an arithmetic operator (+, -, ×, /).
Enter the second number.
Press = to view the result.
Use AC to clear the display or DEL to remove the last entered digit.
